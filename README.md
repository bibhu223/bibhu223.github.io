# bibhu223.github.io

Professional academic website for Dr. Bibhutibhusan Nayak.

## Upload these files to the repository root

- index.html
- style.css
- script.js
- assets/

## Replace your photo

1. Add your real photo as `assets/profile.jpg`
2. In `index.html`, replace:

```html
src="assets/profile-placeholder.svg"
```

with:

```html
src="assets/profile.jpg"
```

## Add your CV

Place your PDF at:

```text
assets/Bibhutibhusan_Nayak_CV.pdf
```

## Important edits in index.html

Replace:

- `YOUR_EMAIL@dgist.ac.kr`
- Google Scholar URL
- LinkedIn URL
- ORCID URL
- Publication details and links
- News items

## Publish

Push or upload all files to the `main` branch of:

```text
bibhu223.github.io
```

GitHub Pages will update automatically.
